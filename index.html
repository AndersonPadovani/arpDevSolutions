<Tecnologias />

JavaScript é uma linguagem fundamental para o desenvolvimento web, permitindo interatividade dinâmica em páginas da web, melhorando a experiência do usuário e viabilizando aplicativos web modernos. O GitHub é uma plataforma de hospedagem de código amplamente utilizada por desenvolvedores. Ele permite o armazenamento, colaboração, controle de versão e rastreamento de projetos de software, facilitando o trabalho em equipe e a contribuição para projetos de código aberto. Java é uma linguagem de programação de alto nível amplamente usada para desenvolver aplicativos diversos. Ela é conhecida por sua portabilidade, segurança e capacidade de criar software escalável e robusto. Python é uma linguagem de programação versátil e de alto nível. É usada para desenvolver aplicativos, scripts e soluções de IA, com sintaxe clara e ampla biblioteca padrão, tornando-a popular entre desenvolvedores e cientistas de dados. React é uma biblioteca JavaScript amplamente usada para desenvolver interfaces de usuário interativas em aplicativos web. Ela permite criar componentes reutilizáveis, facilitando o desenvolvimento de interfaces dinâmicas e responsivas. SQL (Structured Query Language) é uma linguagem de programação usada para gerenciar e manipular bancos de dados relacionais. Ela permite a criação, consulta, atualização e exclusão de dados em sistemas de gerenciamento de banco de dados. O Node.js é um framework que permite a criação de APIs, aplicações de chat e microsserviços. Possui facilidade em seu aprendizado e utiliza o JavaScript para executar essa linguagem pelo lado do servidor. Ele é recomendado para ser utilizado em conjunto com bancos de dados relacionais, tais como MySQL, PostgreSQL.
